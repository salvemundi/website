Architectuur Samenvatting

Door deze strikte scheiding van applicaties bereiken we:

Minimale Permissies: De Provisioning service kan accounts aanmaken, terwijl de Sync service uitsluitend bestaande accounts bijwerkt.

Admin Consent: Voor alle 'Application' permissies is een eenmalige goedkeuring van de Global Admin vereist in de Azure Portal.
